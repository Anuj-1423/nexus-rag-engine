How It Works

From Upload to Answer in 3 Steps

No complex setup. No ML expertise needed. Just upload, ask, and get cited answers.

1

Upload Documents

Drag and drop PDFs, DOCX, or TXT files. Our pipeline auto-chunks, embeds, and indexes them into a vector store in seconds.

2

Ask Anything

Type your question in natural language. FAISS performs semantic vector search to find the most relevant document chunks instantly.

3

Get Cited Answers

The LLM generates a precise, hallucination-free response grounded in your documents with exact source citations.

Architecture

Built on a RAG Pipeline

ENTRAG follows the Retrieval-Augmented Generation pattern, combining the precision of vector search with the fluency of large language models.

Document Store

MySQL metadata catalog with file tracking, versioning, and role-based access policies.

Embedding Engine

Documents are chunked and converted to dense vectors using state-of-the-art embedding models.

FAISS Vector Index

Sub-200ms similarity search across millions of embeddings using Facebook AI Similarity Search.

LLM Synthesis

Retrieved context is sent to your chosen LLM (Llama 3, Mistral, GPT-4) for answer generation with citations.
